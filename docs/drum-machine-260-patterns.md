## Grid Notation

- Each grid represents a one-measure drum pattern.
- Each row = one instrument. An arrow (→) marks instruments used in that pattern.
- Each box = a time unit: **sixteenth note** in 4/4, **eighth note** in 12/8 (triplet feel).
- **Filled** box = hit, **empty** box = rest.
- 4/4 patterns → 16 steps per row. 12/8 patterns → 12 steps per row.
- A **flam** is indicated by "F" preceding the note.

## How To Program a Pattern

1. Select the correct pattern length: 16 steps (4/4) or 12 steps (12/8/triplets).
2. Enter WRITE mode on your drum machine.
3. For each instrument row with an arrow (→), program the hits from left to right.
4. Switch to PLAY mode and listen.
5. Adjust tempo to taste.

## Using the Patterns (Song Mode)

Use **song mode** / **chain mode** to string patterns together:

- Alternate complementary patterns: `1 + 2 + 1 + 2 + 1 + 2 + 1 + Break`
- Or: `1 + 1 + 2 + 2 + 1 + 1 + 2 + Break`
- Breaks go in the last measure of a phrase (typically 8 or 16 measures).
- Substitute **CY** (Ride Cymbal) for **CH** (Closed Hi-Hat) where appropriate.
- Add **CPS** (Claps) or **HT** (High Tom) to accent beats.
